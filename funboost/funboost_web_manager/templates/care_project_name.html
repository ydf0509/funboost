功能说明

care_project_name 用于控制 Funboost Web Manager 只显示某个特定项目的队列信息。

用户是在 @boost(BoosterParams(project_name=$你的项目名字)) 去设置booster是属于什么项目

因为如果多个项目都用一个redis db，或者很多过气遗留或者改过名字的队列元数据在redis中，造成信息干扰和funboost web manager性能下降

设置后的效果：

• 不限制（显示全部）：显示 Redis 中存储的所有队列信息

• 选择具体项目：只显示该项目（project_name）下的队列信息，减少信息干扰，提升性能

⚠️ 注意：设置后会影响本次会话的所有页面（队列操作、消费者信息等），刷新页面后仍然生效，直到手动更改或重启服务。
